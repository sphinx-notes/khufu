Installation
============

Download it from official Python Package Index:

.. code-block:: console

   $ pip install sphinxnotes-khufu

Add extension to :file:`conf.py` in your sphinx project:

TODO

.. code-block:: python

    extensions = [
              # …
              'sphinxnotes.khufu.ext.snippet',
              # …
              ]
