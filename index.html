Installation¶

Download it from official Python Package Index:

$ pip install sphinxnotes-snippet

Add extension to conf.py in your sphinx project:

TODO

extensions = [
          # …
          'sphinxnotes.snippet.ext',
          # …
          ]

Extension¶

Configuration¶

The extension provides the following configuration:

snippet_config

(Type: Dict[str,Any], Default: {}) Configuration of snippet cli command.

snippet_patterns

(Type: Dict[str,List[str]], Default: {})

Command Line Tool¶

A python script named snippet should be installed in your path. (Usually /usr/local/bin or ~/.local/bin).

Invoke snippet --help for usage.
